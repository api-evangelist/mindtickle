# Mindtickle GraphQL API

Mindtickle provides a public GraphQL API for its Call AI platform, offering programmatic access to call recordings, transcriptions, coaching scores, and conversation intelligence data. The REST API covers user provisioning, program management, and reporting; the GraphQL API is scoped to Call AI (conversation intelligence).

**Endpoint:** https://api-gateway.callai.www.mindtickle.com/public/graphapi

**Documentation:** https://www.mindtickle.com/call-ai-public-api-mindtickle/
